# Ledger
Habit tracking web app.


## Environment
- run 
- for bash
```bash 
    python -m venv .venv
    source .venv/bin/active
    pip install -r requirements.txt
```
- windows (cmd prompt)
```CMD 
    python -m venv venv
    venv\Scripts\activate.bat
    pip install -r requirements.txt
```
